# Home Assistant Apps

Custom Apps for Home Assistant.

## Installation

Add this repository to Home Assistant:

1. Go to **Settings** → **Apps** → **Install app**
2. Click menu (⋮) → **Repositories**
3. Add: `https://github.com/jaadth/hass-apps`
4. Install App from the App store
